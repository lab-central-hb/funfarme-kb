# staging/ — Área de entrada de dados brutos

Coloque aqui arquivos brutos (planilhas, CSVs, PDFs, prints, etc.) para o Claude processar e incorporar na base de conhecimento.

## Como usar

1. Copie o arquivo para esta pasta.
2. Peça ao Claude para processar — ex: "processa os arquivos em staging/".
3. Após processado, o Claude move o arquivo para a pasta correta ou o remove.

## Tipos de dados aceitos

- Planilhas de tarefas / atividades (.xlsx, .csv)
- Listas de máscaras / procedimentos
- Inventário de equipamentos
- Registros de incidentes
- Qualquer dado que precise ser incorporado em `docs/`
