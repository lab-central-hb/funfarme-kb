# Equipamentos Laboratoriais

Catálogo completo dos equipamentos do laboratório.

## Urinálise

| Equipamento | Fabricante | Função |
|---|---|---|
| Cobas 6500 I | Roche | Análise de urina com microscopia automatizada |
| Cobas 6500 II | Roche | Segundo analisador de urina com microscopia |

- Backup semanal automatizado das imagens de microscopia.
- Parâmetro "Outros" configurado para importar 6 anormalidades (incluindo Trichomonas).
